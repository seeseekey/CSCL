FTPConnection.DeleteOnFailure Property

Controls whether or not a file is deleted when a failure occurs.

public virtual bool DeleteOnFailure {get; set;}

Remarks

If true, a partially downloaded file is deleted if there is a failure during the download. For example, the connection to the FTP server might have failed. If false, the partially downloaded file remains on the FTP client machine - and the download may be resumed, if it is a binary transfer.

By default this flag is set to true.
